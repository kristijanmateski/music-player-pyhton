# Python Music Player

A simple **console-based music player** written in Python using `pygame`.

It allows you to play MP3 files from a selected folder, with **Pause**, **Resume**, **Stop** and **Shuffle** options.

## Features

- Play MP3 songs from a folder
- Pause, Resume, Stop playback
- Shuffle (play a random song)
- Automatically selects a folder if only one exists or lets you choose if multiple folders are present

## How to Use

- Clone or download this repository.
- Place your MP3 songs in a folder inside the project directory (the same folder where main.py is located)
